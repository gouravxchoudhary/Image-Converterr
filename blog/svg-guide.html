SVG Structure

<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="40" fill="blue"/>
  <rect x="20" y="20" width="60" height="60" fill="red"/>
</svg>

Key Elements

  • svg: Root element with viewBox
  • circle, rect, path: Basic shapes
  • g: Group elements
  • defs: Define reusable elements

Optimization Techniques

1. Clean Up SVG Code

  • Remove unnecessary attributes
  • Simplify path data
  • Use shorter color codes
  • Remove empty groups

2. Minification

Use tools like SVGO to reduce file size: 

# Before optimization: 2.5KB
<svg xmlns="http://www.w3.org/2000/svg">
  <g transform="translate(0,0)">
    <path d="M10 10h100v100h-100z" style="fill: #ff0000;"/>
  </g>
</svg>

# After optimization: 95B
<svg xmlns="http://www.w3.org/2000/svg"><path d="M10 10h100v100H10z" fill="red"/></svg>

Using SVGs on the Web

1. Inline SVG

<div>
  <svg viewBox="0 0 100 100">
    <circle cx="50" cy="50" r="40"/>
  </svg>
</div>

2. As an Image

<img src="icon.svg" alt="Icon description">

3. CSS Background

.icon {
  background-image: url('icon.svg');
  background-size: contain;
}
